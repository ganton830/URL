   ReactNative app

1.Download and install nodejs
2 .choose an Appropriate folder for projects ,open command and type in npm install -g react-native-cli
3.type in command prompt npm install -g yarn
4.create new reactive nativr project with the command react-native init FirstApp
5.To run Your Newly created project ,navigate to the project folder and type in react-native run-android
NB:In case of any errors,Navigate to your ADB Location folder and command adb reverse tcp:8081 tcp:8081
And Return the project.
